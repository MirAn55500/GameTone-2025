# Геймтон DatsJingleBang - Клиент для игры

Интеллектуальный клиент для игры DatsJingleBang с оптимизированной стратегией победы.

## Стратегия победы

Клиент реализует комплексную стратегию для максимизации очков и победы:

### 1. Приоритет на уничтожение препятствий группами
   - **Основной источник очков**: Ищет позиции, где одна бомба может уничтожить несколько препятствий
   - **Максимизация очков**: 1+2+3+4 = 10 очков за взрыв 4+ препятствий (максимум)
   - **Умный расчет**: Учитывает текущий радиус бомбы при планировании
   - **Оптимизация**: Выбирает цели с наибольшим потенциалом очков

### 2. Безопасность превыше всего
   - **Проверка безопасности**: Анализирует все бомбы на карте и их радиусы взрыва
   - **Автоматическое отступление**: При обнаружении опасности находит безопасный путь
   - **Учет времени**: Рассчитывает время движения и проверяет безопасность с учетом этого
   - **Защита от мобов**: Избегает контакта с активными мобами

### 3. Многоуровневая система приоритетов
   Приоритет целей:
   1. **Группы препятствий** - основной источник очков
   2. **Мобы** - 10 очков за убийство (но опасны при контакте)
   3. **Вражеские юниты** - 10 очков за убийство (сложнее попасть)
   4. **Одиночные препятствия** - резервная цель

### 4. Оптимизированная система улучшений
   Приоритет покупки улучшений:
   1. **Радиус бомбы** - КРИТИЧНО! Позволяет уничтожать больше препятствий за взрыв
   2. **Скорость** (до 3 уровня) - для быстрого перемещения и избегания опасностей
   3. **Количество бомб** - для большей эффективности
   4. **Акробатика** (уровень 1) - позволяет проходить через бомбы (безопасность)
   5. **Зрение** - для лучшей видимости и планирования
   6. **Фитиль бомбы** (до 3 уровня) - для быстрых взрывов
   7. **Броня** - защита от случайных взрывов

### 5. Умное планирование пути
   - **Алгоритм A***: Использует оптимальный алгоритм поиска пути
   - **Учет препятствий**: Обходит стены и разрушаемые препятствия
   - **Учет бомб**: Избегает бомб (если нет акробатики)
   - **Оптимизация**: Находит кратчайший безопасный путь

### 6. Дополнительные тактики
   - **Избегание смерти**: Штраф 10% очков при возрождении - критично избегать
   - **Эффективное использование бомб**: Размещает бомбы только когда это выгодно
   - **Координация юнитов**: Управляет всеми 6 юнитами одновременно
   - **Адаптивность**: Учитывает текущие улучшения при принятии решений

## Установка

```bash
pip install -r requirements.txt
```

## Использование

### Базовое использование

```bash
python game_client.py YOUR_API_KEY
```

### Использование тестового сервера

```bash
python game_client.py YOUR_API_KEY https://games-test.datsteam.dev
```

## Получение API ключа

1. Зарегистрируйтесь на платформе геймтона
2. Получите API ключ в личном кабинете
3. Используйте его при запуске клиента

## Особенности

- Автоматическое управление всеми 6 юнитами
- Интеллектуальное размещение бомб для максимизации очков
- Автоматическая покупка улучшений
- Защита от взрывов и опасных зон
- Соблюдение ограничений API (3 запроса/сек)

## Структура проекта

- `game_client.py` - основной файл клиента

- `README.md` - документация

## Дополнительные возможности

Клиент уже включает:
- ✅ Охоту на мобов (10 очков за моба)
- ✅ Атаку вражеских юнитов (10 очков за убийство)
- ✅ Сложную логику планирования с A*
- ✅ Адаптивную стратегию в зависимости от улучшений

Возможные улучшения:
- Предсказание движения врагов
- Более сложная логика групповой атаки
- Оптимизация для достижений (например, цепная реакция)
- Анализ статистики и адаптация стратегии

## Примечания

- Игра работает с частотой дискретизации ~50мс
- Максимум 3 запроса в секунду к API
- При смерти всех юнитов - штраф 10% от очков
- Скилл поинты выдаются каждые 1.5 минуты (максимум 10 за раунд)
